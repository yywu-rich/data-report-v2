# 售后工单数据分析平台取数口径

状态：已接入“一键更新报表”书签。

平台：`http://192.168.212.151:3005`

## 使用页面

1. 选择统计开始日期和结束日期。
2. 打开“TS 数据”页签。
3. 点击页面上的“刷新数据”，等待卡片更新。
4. 点击“一键更新报表”书签。
5. 核对确认框中的日期和数值后写入。

## 写入字段

- “TS 接单量”写入 `secondLineOrders`。
- “周解决率”除以 100 后写入 `secondLineResolveRate`。
- 页面日期按开始日、结束日均包含处理，并以结束日确定报表周。

示例：

```text
页面日期：2026-07-17 至 2026-07-23
TS 接单量：204
周解决率：73.5%
```

写入：

```json
{
  "secondLineOrders": 204,
  "secondLineResolveRate": 0.735
}
```

## 注意

- 仅从页面展示结果取数，不再使用 Jira JQL 计算这两个指标。
- Jira 中保留原二线计算草案，但保持暂停。
- Jira 更新不会覆盖本平台已经写入的二线数据。
- 历史数据需要在平台中逐周修改日期、刷新后再点击书签；不支持一次自动补抓多周。
